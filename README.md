# Kobalt theme

A syntax theme for Atom based on the Cobalt TextMate theme by Jacob Rus.

Get the atom [package](https://atom.io/themes/kobalt). Pairs nicely with the [kobal-ui](https://atom.io/themes/kobalt-ui) theme.
